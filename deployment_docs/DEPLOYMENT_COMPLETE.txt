🚨 EMERGENCY PROCEDURES

Database corrupted?
→ Restore from S3: aws s3 cp s3://bucket/backup.sql.gz - | gunzip | psql -U user -d db

Disk full?
→ Clean old backups: find /backups -mtime +14 -delete

Agent won't start?
→ Check logs: docker-compose logs -f agent
→ Verify DB: docker-compose exec postgres psql -U user -c "SELECT 1"

More details in PRODUCTION_SETUP.md troubleshooting section.
