  CALL RECORDING TOOLS FOR LIVEKIT - START HERE
================================================================================

🎯 WHAT IS THIS?

A complete toolkit to download, analyze, and debug your calling agent using
actual call recordings from LiveKit Cloud.

================================================================================
  5-MINUTE QUICK START
================================================================================

1. Verify Your Setup
   $ cat ../.env | grep LIVEKIT
   # Should show: LIVEKIT_URL, LIVEKIT_API_KEY, LIVEKIT_API_SECRET

2. Download Recent Calls
   $ python download_livekit_recordings.py --recent 5
   # Creates: recordings/ folder with .webm files

3. Analyze Audio Quality
   $ python analyze_call_quality.py recordings/*/*.webm
   # Shows: Quality score (0-100), audio metrics, issues

4. Listen to Recording
   $ python playback_recordings.py --play recording.webm
   # Plays using VLC, mpv, or ffplay

5. Results
   - Quality ≥ 80? Good to deploy!
   - Quality < 80? Debug before deploying

List all recordings:

  python download_livekit_recordings.py --download room_name

Analyze quality:
  python analyze_call_quality.py recordings/room/file.webm

Show storage used:
  python playback_recordings.py --storage

================================================================================
  QUALITY SCORE GUIDE
================================================================================

Quality Score Range:
  ✅ 90-100  Excellent    → Ready to deploy
  ✓  70-89   Good         → Monitor, acceptable
  ⚠️  50-69   Fair         → Investigate issues
  ❌ <50     Poor         → Debug before using

What Affects Quality:
  • Sample rate (should be 24000 Hz)
  • Bitrate (should be 128+ kbps)
  • Channels (should be 2 - stereo)
  • Audio clarity (no cutting/popping)

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "No recordings found"
  → Check .env: LIVEKIT_URL, LIVEKIT_API_KEY, LIVEKIT_API_SECRET

Problem: "ffprobe not found"
  → Install ffmpeg: brew install ffmpeg

Problem: "No audio player found"
  → Install VLC: brew install vlc

Problem: "Permission denied"
  → Run: chmod +x *.py

Problem: "Module not found (httpx)"
  → Install: pip install httpx python-dotenv
